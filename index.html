YouTube Video Chat Assistant

An AI-powered Chrome Extension that enables users to chat with any YouTube video's transcript. It uses Retrieval-Augmented Generation (RAG) with LangChain, FAISS, and Google Gemini Embeddings to deliver context-aware answers based solely on the video's content.

Key Features

  • ✔ Chat with any YouTube video's transcript
  • ✔ RAG pipeline with semantic search using FAISS
  • ✔ Context-aware responses powered by Google Gemini
  • ✔ Chrome Side Panel Extension with custom HTML/CSS/JS UI
  • ✔ FastAPI backend with automatic transcript extraction
